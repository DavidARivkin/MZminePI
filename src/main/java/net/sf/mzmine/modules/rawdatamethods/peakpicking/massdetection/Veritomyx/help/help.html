Veritomyx PeakInvestigator™ Peak Detection Software for Metabolomics

Updated: January 21, 2014

Description

Mass spectral raw and processed results often fail to deconvolve and consider crucial data. This contributes to inaccurate, incomplete and inefficient protein and metabolite identifications. Veritomyx, Inc. mass spectral analysis software applies advanced signal processing and identification algorithms to reveal hidden data and accelerate to correct biomolecular identifications. This improves R&D effectiveness while saving time and money.

Critical data remain hidden when overlapping peaks are unresolved. Unresolved peaks cause:

  • Inaccurate IDs
  • Misdirected efforts
  • Wasted time and resources
  • Missed biomarkers

PeakInvestigator™ mass spectral analysis software detects and precisely deconvolves hidden overlapped peaks, accelerating you to confident molecular identifications in your metabolomics workflow. PeakInvestigator doubles the effective resolution of virtually any existing mass analyzer. Unleashing important discoveries that would have remained hidden in your data offers critical leverage for scientific and business success.

Method Parameters

The only parameters required for using this module are the Veritomyx SaaS credentials that you can enter into the project preferences. The other parameters set in this dialog box define the MZmine behavior and are explained in other locations in the MZmine documentation.

Veritomyx Username (from MZmine Prefrences)
Login name (email address) for Veritomyx SaaS.
Veritomyx Password (from MZmine Prefrences)
Password for access to Veritomyx SaaS.
Veritomyx Project Number (from MZmine Prefrences)
Veritomyx project number under which this job is to be run. This number is intended as an aid for tracking multiple projects and users and it is assigned along with your login credentials. A single login can have access to one or more projects. Your Veritomyx login must have access to the project number you enter here. Setting this to zero will use your default project.

Processing Sequencing

This is the process for using the Veritomyx PeakInvestigator module.

  1. Load a raw data file. Once you select the file or a set of scans (as shown) you are prepared to execute the process either by right-clicking on the raw data file as in the first figure or by selecting the menu as shown in the second figure.





  2. Once you have executed the Peaks/mass detection module, the following dialog will appear.



  3. Select PeakInvestigator in the pulldown box and give the mass list a name. The selection of MS level is only applicable when you have selected the whole raw data file for processing. In this example we provided a set of scans (which may, but doesn't have to, include scans of different MS level) to process so the MS level is ignored. MS level is defined in the other MZmine documentation. Basically, an MS scan is MS level 1, whereas a tandem MS scan is MS level 2, etc.
  4. While the job is being prepared to launch, the following task window will appear.


  5. Once the job is launched, the entry in the "Tasks in progress..." box will disappear and you will see the job appear under the raw data file. At this point, the job has been sent to the Veritomyx servers for processing which can take some time depending on the number of scans submitted. At this point you can optionally save the MZminePI project and come back later. When you load the saved project, it will bring you back to this same point. Be careful not to close MZminePI without saving the project because any unsaved work will be lost, including the info about the running job which will make it very difficult to retrieve that data later.



  6. Now right click on the job to download the results from the Veritomyx servers. If the job is not complete yet, this will wait for the job to finish and then download and install the resulting peaks files.



  7. Once the job results are downloaded, the peaks lists are loaded under each of the scans originally selected.
